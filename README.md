# Blossom
## 目的
Blossomは主に英単語の学習を目的としています。
が、英単語以外の学習も可能です。

## 特徴
自由度の高いインターフェースや分単位で指定できる2つの復習機能により、自分に合った勉強のプランを実行するのに役立ちます。
